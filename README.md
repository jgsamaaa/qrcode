# QR Code Generator

A simple ReactJS with Tailwindcss that can generate a QR Code, you can download the QR Code image as well.

Live Server: https://qrcodegeneratorproject.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/jgsamaaa/React-QR-Code-Generator.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
